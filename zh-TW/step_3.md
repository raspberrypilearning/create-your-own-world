## 堅固的牆壁

\--- task \---

再測試你的`玩家`角色。 你觀察到了嗎？它可以穿過淺灰色的牆壁…這一點也不科學！

![截圖](images/world-walls.png)

\--- /task \---

\--- task \---

要解決這個問題，你必須讓`玩家`角色在碰到淺灰色牆壁時，讓它彈回來。 請把這個程式添加到`重複無限次`{:class="block3control"}區段中面朝方向積木的下面：

![玩家](images/player.png)

```blocks3
當 @greenflag 被點擊
重複無限次
    如果 <(向上 v) 鍵被按下？ > 那麼
        面朝 (0) 度
        移動 (4) 點
    end
    如果 <(向左 v) 鍵被按下？ > 那麼
        面朝 (-90) 度
        移動 (4) 點
    end
    如果 <(向下 v) 鍵被按下？ > 那麼
        面朝 (180) 度
        移動 (4) 點
    end
+    如果 <(向右 v) 鍵被按下？ > 那麼
        面朝 (90) 度
        移動 (4) 點
    end
+  如果 <碰到顏色 (#BABABA)？ > 那麼
    移動 (-4) 點
    end
end
```

\--- /task \---
